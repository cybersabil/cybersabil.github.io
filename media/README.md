# Media Folder

Purpose: keep all website images and uploadable media in one safe place for GitHub Pages and Pages CMS.

Recommended usage:

- Put social preview images here, for example `media/social-preview.png` or `media/social-preview.svg`.
- Put portfolio project screenshots here, for example `media/project-trafficmonitor.png`.
- Use relative paths in JSON, for example `media/project-trafficmonitor.png`.
- Compress large screenshots before uploading. Prefer 1200px wide or smaller for normal website cards.
- Do not store private files, passwords, tokens, license keys, customer data or admin-only scripts here because GitHub Pages files are public.

Current default previews:

- `media/social-preview.png` for broad social-platform compatibility.
- `media/social-preview.svg` as the editable vector source.
