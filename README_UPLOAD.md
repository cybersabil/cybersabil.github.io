# CyberSabil Final Consolidated Package

This folder contains the complete GitHub Pages project after cumulative CMS field, collision, responsive, malformed-data, link-safety, build-generation and control-isolation corrections.

## Upload

Open this folder and upload **its contents** to the repository root. Same-name files must replace the existing files and new audit/test files may be added.

Do not create this incorrect structure:

```text
repository/cybersabil_final_consolidation/index.html
```

The correct structure is:

```text
repository/index.html
repository/assets/
repository/data/
repository/tools/
```

After upload, wait for the custom `Build and Deploy CyberSabil` GitHub Action to pass before testing Pages CMS or the live site.
